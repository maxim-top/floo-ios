# BMXRoomPubConfigureOptions Class Reference

  **Inherits from** NSObject  
  **Declared in** floo_proxy.h  

## Overview

@interface 房间配置参数
mEnableAudio 是否启用音频
mEnableVideo 是否启用视频
mEnableData 是否启用数据
mDisplay 用户描述
mWidth 视频宽度
mHeight 视频高度
mBitrate 视频帧率
